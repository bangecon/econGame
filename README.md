# econGame
Package: econGame

Title: Tools for in-class game experiments for Principles of Economics

Version: 0.0.1

Author: James Bang  <bangecon@gmail.com>    

Description: This package pulls activity data from Google Forms submissions to a 
    Google Sheets file; tabulates, scores, and graphs the responses; and 
    writes the grade data to a separate Google Sheet. Supported games include: 
    
      1.  Market equilibrium game; 
      2.  Entry and exit game; 
      3.  Bertrand game;
      4.  Cournot game; 
      5.  Stackelberg game.
      
 All documentation files include links to Google Forms addresses that can be 
copied and saved. 

Depends: 

  * R(>= 4.0.0),  
  * googlesheets4,
  * googledrive,
  * ggplot2, 
  * ggpubr, 
  * tidyr, 
  * dplyr, 
  * gtsummary, 
  * stringr
